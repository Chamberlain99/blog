# 爬取数据

## 需要先安装requests 数据请求模块

- `pip3 install requests`

## 需要安装界面美化工具

- `pip3 install beautifulsoup4`

## 导入excel 需要 xlwt xlrd 插件

- `pip3 install xlwt` 写入数据
- `pip3 install xlrd` 读取数据
